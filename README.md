## Overview
This project is a personal website built as part of **School assignment**.  
The goal of the lab is to learn the fundamentals of front-end web development by creating a multi-page website using **HTML** and **CSS**, and enhancing the design using **Bootstrap 5.3**.

The website focuses on visual presentation only. Future labs will extend this project with JavaScript, backend logic, databases, and APIs.

---

## Learning Objectives
This project addresses the following learning objectives:

- **LO1:** Build websites using HTML  
- **LO2:** Organize and present content using HTML  
- **LO3:** Apply CSS styling using a pre-built library  
- **LO4:** Use a CSS library not explicitly taught in class (Bootstrap)
